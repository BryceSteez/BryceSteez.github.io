Spanish Website Mockup for web design

This is a mockup for a big project we did for web design class my sophomore year, this was the desktop version of the website. The challenge I encountered with this website was that the user had to be able to switch back and forth between spanish and english. Another problem was after my first design, a certain group member did not agree with my designs. He thought my first designs were too hard to code and the colors didn't make the website look friendly. Just because of this, my group made me redo all of the mockups. So we ended up way behind schedule because my group made me redo the whole website. My group depended on my designs for the website to be created and for me to have a mockup in english and spanish and remake all of the mockups. It made me have to work double time just to get it done so that they have enough time to code the website. Although there were a lot of complications, I was able to finish the mockup successfully.
