Winter Destination Mock Up and Website

This was for a web design project in my junior year of high school. We had to create websites that work on the computer and mobile devices. I used Adobe Photoshop for the mockup, and Brackets for coding. The point of this website was to showcase a winter-related hotel. I made the design so that it would have a feel of winter. This project taught me how to design and code for mobile and tablet. It also taught me how to make a one page website with all the content on one page. The website actually didn’t come out exactly like the mockup . At the time, it was too advanced for me to code exactly what I intended.
