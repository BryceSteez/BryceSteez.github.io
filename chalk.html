Chalkboard Underage Drinking

The “Chalkboard Underage Drinking” design was a PSA to discourage underage drinking.I created this using Adobe Photoshop. This was a contest in my Web Design class in my junior year of high school where we had to have a graphic and a video for the PSA. We were able to get in the top 50 of all the submissions. I learned about all the consequences that came with underage drinking. I also learned some new skills in Photoshop like using smart objects and such.
